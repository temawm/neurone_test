### Писал синглмодуль/активити, тесты не писал, комментировал по максимуму, немного отошел от дизайна, но в целом все также.
В коммитах есть версия без вынесения строк в ресурсы.

Тестировал только на средний размер экрана, горизонтально/вертикально, основной ноут в сервисе, а этот эмулятор не тянет, поэтому, могут быть артефакты.

Есть core слой, в нем все базовое, application класс, темы, shared-компоненты, навигация.

Тему использовал композовскую, при желании можно сразу докрутить вторую.

* **MVVM + Clean**
* **Kotlin 2.0.21**
* **Jetpack Compose** 
* **Navigation Compose**
* **Koin 3.5.6**
* **Kotlin Coroutines 1.8.1**
* **Kotlinx Serialization 1.7.3**
* **Compose BOM**
* **Lifecycle Runtime KTX**
* **Activity Compose**
